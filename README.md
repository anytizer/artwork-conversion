# artwork-conversion
A communication tool for customer, developer and admin - to convert an artwork into website or software.

# Installation
set the following subdirectories into your subdomains:
 * admin/public_html
 * api/public_html
 * assets/public_html
 * customers/public_html
 * developers/public_html
 * hooks/public_html
 * www/public_html

# Database
